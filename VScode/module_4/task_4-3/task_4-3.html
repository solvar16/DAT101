  4. [Explain the main difference between DOM id and DOM name. Can we have identical IDs?]

    5. [DOM id er en unik identifikator for et HTML-element på en side. Hvert element må ha en unik id innenfor samme dokument, og vi kan bruke id til å referere til elementet i JavaScript. ID-er må være ulike på en side. Hvis flere elementer har samme id, kan det føre til problemer med JavaScript og CSS. DOM name brukes på skjemafelt for å gruppere elementer sammen, og flere elementer kan ha samme name. Det er ikke nødvendig at name er unik.]
